你可以直接增加或者修改位于configure文件夹下的ini文件.

我们使用的是ini配置文件

请使用database作为section名字来指示数据库的连接，在这个section下，您需要配置：

名称 填充内容 备注
type 数据库类型 暂时只支持MySQL和SQLite
user 数据库用户 请尽可能选用具有最小的privileges的用户来访问这个数据库
password 数据库密码 SQLite请留空
host 数据库地址 本地可选择localhost
name 数据库名称 SQLite不用给上后缀名.db
这是我们给的一个实例
[database]
type = MySQL
user = root
password =
host = localhost
name = npupt
